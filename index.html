☁ Google Drive Backup
How to sync with Google Drive:
  1. Click ☁ Export → Drive in the header — your data downloads as a .json file.
  2. Open drive.google.com and upload the file to a folder of your choice (e.g. PSX Backups).
  3. To restore on any device: download the file from Drive, click 📂 Import ← Drive, paste the file contents.

Auto-sync tip: If you have Google Drive desktop app installed, export directly into your Drive sync folder — it uploads automatically.
